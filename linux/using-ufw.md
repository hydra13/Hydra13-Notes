# Использование Uncomplicated Firewall (ufw)

---------------------------------------
* разрешаем входящий трафик с 25-й подсети:
```
	sudo ufw allow from 192.168.25.0/24
```
* заврещаем весь входной трафик от остальных подситей:
```
	sudo ufw deny from any to any 
```
* запускаем:
```
	sudo ufw enable
```
* проверяем:
```
	sudo ufw status
```
